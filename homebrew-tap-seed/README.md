# pixelicous/homebrew-vedox

Homebrew tap for [Vedox](https://github.com/pixelicous/vedox) — local-first, Git-native documentation operating system for solo developers.

## install

    brew tap pixelicous/vedox
    brew install vedox

## daemon (optional — auto-start on login)

    brew services start vedox

## uninstall

    brew services stop vedox
    brew uninstall vedox
    brew untap pixelicous/vedox
